# Review Checklist

## Code Quality
- [ ] RuboCop passing
- [ ] No TODO comments left
- [ ] No debug code (puts, binding.pry)
- [ ] Consistent naming conventions

## Testing
- [ ] All specs passing
- [ ] Coverage > 80%
- [ ] Integration tests passing
- [ ] Edge cases covered

## Documentation
- [ ] README complete
- [ ] API documented
- [ ] Events documented
- [ ] Setup instructions work

## Security
- [ ] No secrets in code
- [ ] Environment variables used
- [ ] Input validation present

## Performance
- [ ] Database indexes added
- [ ] N+1 queries avoided
- [ ] Connection pooling configured
